# Changelog

## 18.0.1.0

### Initial Release
- Odoo 18 Community Edition
- Built-in PostgreSQL 16 (no pgvector - optimized for edge hosts)
- Built-in Nginx reverse proxy with WebSocket support
- HTTP-only for LAN access (use Cloudflare Tunnel for HTTPS)
- s6-overlay process management
- Configurable workers, memory limits, and timeouts
- Traditional Chinese (繁體中文) translations
- Home Assistant add-on compatible
